Progetto del corso Basi di dati (cod. 861II) del Corso di Laurea in Ingegneria Informatica.

Si desidera progettare un database relazionale su DBMS Oracle MySQL che permetta di memorizzare i dati relativi a un sistema che gestisce film, FilmSphere. 
FilmSphere è un servizio dedicato alla memorizzazione e allo streaming di contenuti video online. Oltre a ospitare i dati, il database sarà dotato di alcune operazioni e funzionalità di data analytics.

Autori: Tommaso Pellegrini e Niccolò Franchi
